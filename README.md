# ONGC Intelligent Knowledge Assistant

An AI-powered enterprise knowledge assistant developed during the ONGC Summer Internship.

## Objective

To develop a Retrieval-Augmented Generation (RAG) based chatbot capable of answering questions from ONGC-related documents including:

- Safety Manuals
- HR Policies
- Technical Documentation
- Operational Procedures
- Internship Resources

## Tech Stack

- Ollama
- Llama 3
- OpenWebUI
- ChromaDB
- LangChain
- FastAPI
- Python

## Features

- Local LLM Inference
- RAG Pipeline
- Multi-Domain Knowledge Base
- User Feedback Collection
- Query Logging
- Knowledge Retrieval from Documents
